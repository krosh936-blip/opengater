# Opengater — личный кабинет

Фронтенд на Next.js для личного кабинета Opengater. В проекте есть прокси к API, авторизация (email/Telegram), локализация и переключение валюты.

## Запуск
1. Установить зависимости: `npm install`
2. Запустить dev-сервер: `npm run dev`
3. Открыть: `http://localhost:3000`

## Конфигурация
Быстрые настройки находятся в `lib/appConfig.ts`. Краткая памятка — в `CONFIG.md`.

Основные пункты:
- профили API и переключение апстримов;
- Telegram‑бот и OAuth‑ссылка;
- домен окна авторизации (popup).

## Авторизация
Используются два набора токенов:
- `user_token` — основной токен для API (через `/api/proxy`).
- `auth_access_token`/`auth_refresh_token` — токены auth‑сервиса (через `/api/auth`).

Если токен невалиден или сервер возвращает 401/403/5xx, пользователь отправляется на `/auth/login`.

## Про временные файлы
В проекте временные файлы не используются. Любые html/js-снимки лучше держать вне репозитория, чтобы не мешали сборке.
